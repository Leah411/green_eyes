# הגדרת מיקומים (ערים, ישובים, קיבוצים)

## ✅ מה שהושלם

1. **מודל Location** - נוצר מודל חדש למיקומים
2. **Migration** - ה-migration נוצר והופעל
3. **API Endpoint** - `/api/locations/` זמין
4. **טעינת מיקומים** - 41 מיקומים נטענו (דוגמה)
5. **עדכון פרונטאנד** - עמוד הבית משתמש במיקומים

## 📋 רשימת מיקומים שנטענו

נטענו 41 מיקומים כולל:
- ערים מרכזיות (תל אביב, ירושלים, חיפה, וכו')
- ישובים (רמת השרון, גבעת שמואל, וכו')
- קיבוצים (דגניה, עין גדי, וכו')
- מושבים (נהלל, כפר מנדא)

## 🔧 הוספת מיקומים נוספים

### אפשרות 1: דרך Django Admin
1. היכנס ל: http://localhost:8000/admin/
2. לחץ על "מיקומים" (Locations)
3. לחץ "הוסף מיקום"
4. מלא את הפרטים:
   - שם (אנגלית)
   - שם עברית
   - סוג (עיר/ישוב/קיבוץ/מושב)
   - אזור

### אפשרות 2: דרך Command
```powershell
venv\Scripts\Activate.ps1
python manage.py load_locations
```

### אפשרות 3: עדכון הקובץ
ערוך את `core/management/commands/load_locations.py` והוסף מיקומים לרשימה `ISRAELI_LOCATIONS`.

## 📝 פורמט הנתונים

```python
('שם בעברית', 'Name in English', 'location_type', 'אזור')
```

**סוגי מיקומים:**
- `'city'` - עיר
- `'town'` - ישוב
- `'kibbutz'` - קיבוץ
- `'moshav'` - מושב

**אזורים:**
- צפון
- מרכז
- דרום
- חיפה
- ירושלים

## 🔗 API Endpoints

### רשימת כל המיקומים
```
GET /api/locations/
```

### סינון לפי סוג
```
GET /api/locations/?type=city
GET /api/locations/?type=town
GET /api/locations/?type=kibbutz
GET /api/locations/?type=moshav
```

### חיפוש לפי שם
```
GET /api/locations/?search=תל
```

## 📊 שימוש בפרונטאנד

הפרונטאנד משתמש ב-API באופן אוטומטי:
- עמוד הבית טוען את כל המיקומים
- המשתמש בוחר מיקום מהרשימה
- המיקום נשמר בדוח הזמינות

## 🎯 מה הלאה?

אם תרצה להוסיף את כל הערים והישובים בישראל (כ-1200+ מיקומים), תוכל:

1. **לייבא מקובץ CSV** - יצירת command שייבוא מקובץ CSV
2. **לייבא מ-API חיצוני** - שימוש ב-API של הלשכה המרכזית לסטטיסטיקה
3. **להוסיף ידנית** - דרך Django Admin

המערכת מוכנה לקבל כל מספר של מיקומים!
